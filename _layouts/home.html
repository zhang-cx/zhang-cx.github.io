    Baxter Fruit Picking

    • Our project uses a Baxter machine to identify and selectively retrieve fruit from a table for storage in a packing bin. We find our project interesting because it achieves successful grasping of desired objects in a scene without having to input information about the shape or pose of the object. Although we used fruits as a simplifying scenario, we believe our system to be adaptable to a more general set of objects given more hardware and minor modification of the grasp and detection subsystems. We believe our work to be useful in situations where particular items need to be selectively retrieved among a greater collection of objects. One such example may be in a production line, where various items of different type need to selected for processing/packaging.
